Project Overview

Project Objective

The primary goal of this project is to develop an automated solution for public safety by monitoring crowd density in real-time. By utilizing Artificial Intelligence and Computer Vision, we aim to reduce the reliance on manual supervision and provide an early warning system for high-risk, overcrowded situations.

Technologies Used

Python OpenCV YOLO Model Flask Backend HTML5/CSS3 JavaScript

Real-World Applications

Railway Stations

Monitor overcrowding on platforms and at ticketing counters for safe passenger flow.

Shopping Malls

Identify peak hours and high-traffic corridors to optimize security and space usage.

Stadiums & Festivals

Manage large gatherings during events to prevent stampedes and ensure emergency exit safety.
